Wanting to expand game development, I came across the JavaScript framework, Phaser. Phaser is a 2d visual Javascript framework used for making HTML games. Animations are able to be rendered and coded much quicker than with vanilla javascript. I figured that once game animations and conditionals became more complex for my final game, I need a new, quick system for development. Using this framework, I created a Geometry Dash clone, called Geometry Crash, which mirrors the physics of former game. The project utilizes global timings, special collision events, and custom builder functions.
